# MSInfo

![image](https://github.com/UnitySio/MSInfo/assets/38973547/6864dbfa-355e-43aa-a757-6c5e33024cd2)

메이플스토리의 Open API를 활용하여 각종 정보를 확인할 수 있게 만든 프로그램입니다.

### 사용 언어
* C++

### 사용 컴파일러
* VC++

### 사용 API
* NEXON OPEN API

### 사용 라이브러리
* 그래픽 - DirectX 11
* UI - ImGui
* 데이터 통신 - Curl
* JSON - RapidJSON
* 이미지 처리 - STB
